# Introduction

# **Purpose**

The purpose of this system is to develop a Collaborative Coding Interview Practice Platform that
enables users to practice coding problems in a shared, real-time environment. The platform aims
to simulate technical interview scenarios by allowing multiple users to collaborate, communicate,
and solve problems together.

# **Problem Statement**

Current coding practice platforms (e.g., LeetCode) are primarily designed for individual use
However, real-world technical interviews often involve collaboration, discussion, and real-time
problem solving.
Additionally, users currently rely on multiple tools (e.g., video calls + code editors), which creates
inefficiency.
This system solves the problem by:
• Providing real-time collaborative coding
• Integrating communication and coding in one platform
• Enabling mock interview environments

# **Scope**

The system is a web-based application that allows users to:

Register and log in

Create or join coding rooms

Collaborate on code in real-time

Execute code and view output

Communicate via chat

Track performance through statistics and streaks

The system does not include voice or video communication and focuses on text-based interaction.
